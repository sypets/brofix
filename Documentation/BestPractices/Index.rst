==============
Best Practices
==============


#. It is recommended to execute a regular full check (e.g. once a day) via the
   console command to make sure the broken links are mostly up to date.

#. Reduce the number of false positives by using the "Exclude"
   button, see :ref:`howItWorksExcludeLinkTargets`.
