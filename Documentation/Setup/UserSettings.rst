================
BE User Settings
================

BE user settings in the core is explained here: https://docs.typo3.org/m/typo3/reference-coreapi/main/en-us/Configuration/UserSettingsConfiguration/Index.html

The following settings are added for brofix:

*  Show message in page module if broken links exist on page *[tx_brofix_showPageCalloutBrokenLinksExist]* in tab "Broken links"

   *  default: on
